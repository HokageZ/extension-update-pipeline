HungerStation Fraud Detection

Install the extension as an unpacked extension. Updates are handled automatically from inside the extension once the updater helper is installed.

Why unpacked? Modern Chrome, Edge, and Brave disable off-store extensions installed via enterprise policy on personal (non-domain-joined) computers. Loading the extension unpacked avoids this restriction while the built-in updater keeps it current.
  1. Download the latest extension.zip below and extract it to a folder on your computer (for example, C:\Users\You\fd-extension).
  2. Open chrome://extensions (or edge://extensions / brave://extensions).
  3. Enable Developer mode using the toggle in the top-right corner.
  4. Click Load unpacked and select the extracted extension folder.
  5. Download and run install-updater.bat as Administrator. This registers the helper that the extension uses to download and install future updates.
  6. Open the extension dashboard, go to the Help & Updates tab, and click Check for Updates to verify everything works.
