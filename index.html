HungerStation Fraud Detection

Install the signed browser extension and run the policy helper to keep it enabled.

Important: You must fully restart Chrome after running the policy helper. Close every Chrome window, end any remaining chrome.exe processes in Task Manager, then re-open Chrome. Otherwise Chrome will keep disabling the extension.
  1. Download and run install-policy.bat as Administrator (right-click → Run as administrator). This tells Chrome to allow the extension.
  2. Fully restart Chrome. The extension may auto-install after the restart. If it does not, continue to step 3.
  3. Download extension.crx, go to chrome://extensions with Developer mode on, and drag the file onto the page.
