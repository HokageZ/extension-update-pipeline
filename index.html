One-click installer for the unpacked extension. Downloads the latest build, registers the auto-updater, and opens your default browser ready to load the extension.

Why unpacked? Modern Chrome, Edge, and Brave disable off-store extensions installed via enterprise policy on personal (non-domain-joined) computers. Loading the extension unpacked avoids this restriction while the built-in updater keeps it current.
  1. Click the Download install.bat button below.
  2. Right-click the downloaded file and choose Run as administrator.
  3. The installer will detect your default browser, download the latest extension, register the updater, copy the extension path to your clipboard, and open the browser.
  4. In the browser window that opens, enable Developer mode, click Load unpacked, press Ctrl+V, then click Select Folder.
  5. Done! The extension will now check for updates and prompt you to install them from the Help & Updates tab.
