HungerStation Fraud Detection

Setup instructions. The installer already copied the extension path to your clipboard.

1Enable Developer mode

In the Extensions tab that opened, toggle Developer mode to ON (top-right corner).

2Load the extension

  1. Click Load unpacked.
  2. Press Ctrl+V in the folder path field, then press Enter.
  3. Click Select Folder.
The path is already copied. It looks like:
C:\Users\<you>\AppData\Local\HungerStation-FD\extension

3Pin the extension

Click the puzzle icon in the browser toolbar, find HungerStation FD, and click the pin so the dashboard is one click away.

4Open the dashboard

Click the pinned icon and select Open Dashboard. Use the Help & Updates tab to check for updates at any time.
